# LuxScape Motors Website

This repository contains the code for the LuxScape Motors website. The website showcases luxury vehicles and allows users to explore different car models, view their specifications, and make reservations.

## Files Included

- `index.html`: The main HTML file that contains the structure of the website.
- `style.css`: The CSS file that contains the styles for the website layout and design.
- `script.js`: The JavaScript file that contains the functionality for the website, including updating car information and handling reservations.

## Features

- **Navigation**: The website features a navigation bar that allows users to browse different car models by clicking on their respective links.
- **Dynamic Content**: When a user clicks on a car model, the website dynamically updates to display the specifications and rental information for that model.
- **Reservation System**: Users can reserve a car by clicking on the "Reserve Now" button, which prompts them to enter their name and email address. Upon successful reservation, a confirmation message is displayed.

## Technologies Used

- **HTML**: Used for creating the structure of the website.
- **CSS**: Used for styling the layout and design of the website.
- **JavaScript**: Used for adding interactivity and functionality to the website, such as updating car information and handling reservations.

## Usage

To view the website locally, simply open the `index.html` file in a web browser.
